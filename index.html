Introduction

The Domain Name System (DNS) plays an important role in the infrastructure of the Internet. It is “a hierarchical and decentralized naming system for computers, services, or any resource connected to the internet or a private network” (Netguru, n.d.). Known colloquially as “the phonebook of the Internet,” DNS is mainly responsible for translating domain names to IP addresses (Cloudflare, a, n.d.). This website discusses the history of DNS, DNS mechanics, the security challenges posed by DNS, and future enhancements to DNS.
